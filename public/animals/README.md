# Animal Images

Place your animal image files (PNG format) in this folder.

## Required Files:

1. badak jawa.png
2. bakantan.png
3. burung cendrawasih.png
4. gajah.png
5. harimau sumatra.png
6. kancil.png
7. kukang.png
8. orang utan.png

## Image Guidelines:

- **Format**: PNG (recommended for transparency)
- **Size**: Approximately 500x500px or larger
- **Quality**: High quality, clear images
- **Background**: Transparent or white background preferred
- **Style**: Colorful, kid-friendly illustrations or photos

## Note:

The file names must match exactly (including spaces and lowercase) as configured in `/lib/animals.ts`.

If you want to use different names or add more animals, update the configuration in `/lib/animals.ts`.
